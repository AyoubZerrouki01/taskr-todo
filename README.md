## ✨ Fonctionnalités

- ✅ Ajouter, modifier, supprimer des tâches
- 🎯 Priorités : Haute / Moyenne / Basse
- 📂 Catégories personnalisées (`#travail`, `#perso`...)
- 🔍 Filtres dynamiques
- ☀️ Mode Light / 🌙 Dark avec switch
- 🖱️ Drag & Drop pour réordonner les tâches
- 📊 Stats en temps réel + barre de progression
- 💾 Sauvegarde automatique (localStorage)
- ⏰ Temps relatif ("il y a 5m")

## 🚀 Utilisation

Aucune installation requise. Ouvre simplement le fichier dans ton navigateur :

```bash
# Clone le repo
git clone https://github.com/ayoubzerrouki/taskr-todo.git

# Ouvre le fichier
cd taskr-todo
start todo-premium.html
```
